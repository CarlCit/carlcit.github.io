我进控制台依次执行了
`ql check`
`touch /ql/db/app.db && pm2 restart all`
`ql update`
然后就可以进去了，仅供参考。

# 查看登录账号密码
cat /ql/config/auth.json

# 更新并重启青龙
ql update                                              
# 运行自定义脚本extra.sh
ql extra                                               
# 添加单个脚本文件
ql raw <file_url>                                       
# 添加单个仓库的指定脚本
ql repo <repo_url> <whitelist> <blacklist> <dependence> <branch>   
# 删除旧日志
ql rmlog <days>                                        
# 启动tg-bot
ql bot                                                 
# 检测青龙环境并修复
ql check                                               
# 重置登录错误次数
ql resetlet                                            
# 禁用两步登录
ql resettfa
